# AVR Fan Controller System

> **Educational Project Notice**
> This project was developed as an educational embedded systems application using AVR microcontrollers and Embedded C programming. The system demonstrates temperature monitoring, PWM motor control, ADC interfacing, sensor integration, and LCD visualization in a real-time embedded environment.

## Overview

The AVR Fan Controller System is a smart temperature-based fan control application implemented using an AVR microcontroller.

The system automatically:

* Reads ambient temperature using an LM35 temperature sensor
* Displays temperature and fan status on an LCD
* Controls fan speed dynamically using PWM
* Adjusts motor behavior according to temperature ranges

The project demonstrates real-world embedded systems concepts including:

* ADC interfacing
* PWM motor speed control
* Sensor integration
* LCD interfacing
* GPIO configuration
* Modular embedded firmware architecture
* Real-time control systems

The implementation was developed in Embedded C using low-level AVR register programming.

# System Architecture

The project follows a modular driver-based architecture.

| Module           | Responsibility                        |
| ---------------- | ------------------------------------- |
| ADC Driver       | Reads analog sensor values            |
| LM35 Driver      | Converts ADC values to temperature    |
| PWM Driver       | Generates PWM signals for fan control |
| DC Motor Driver  | Controls motor direction and speed    |
| LCD Driver       | Displays system information           |
| GPIO Driver      | Configures and controls AVR pins      |
| Main Application | Coordinates system behavior           |

# Temperature Monitoring

The system reads temperature values using the:

```text
LM35 Temperature Sensor
```

The LM35 outputs an analog voltage proportional to temperature.

The ADC module converts this analog signal into a digital value, which is then processed into:

```text
Temperature in Celsius
```

Example functionality:

```c
LM35_getTemperature()
```

---

# Fan Speed Control Logic

The fan speed changes automatically according to the measured temperature.

### Example Control Logic

| Temperature Range | Fan State | Fan Speed |
| ----------------- | --------- | --------- |
| Below 30°C        | OFF       | 0%        |
| 30°C - 59°C       | ON        | 25%       |
| 60°C - 89°C       | ON        | 50%       |
| 90°C - 119°C      | ON        | 75%       |
| 120°C and Above   | ON        | 100%      |

The fan speed is controlled using PWM signals generated by Timer0.

### PWM Features

| Parameter         | Description           |

# How to Build and Run

## 1. Clone the Repository

```bash
git clone https://github.com/MahmoudEhabMorsy/Fan-Controller.git
```

## 2. Open the Project

Import the project into:

* Eclipse CDT
* Atmel Studio
* Microchip Studio

## 3. Compile the Project

Using AVR-GCC:

```bash
avr-gcc Fan_Controller.c adc.c PWM0.c DcMotor.c lcd.c gpio.c lm35_sensor.c -o fan_controller.elf
```

## 4. Flash to AVR Microcontroller

Upload the firmware using:

* USBasp
* AVRDUDE
* Atmel Programmer

## 5. Connect Hardware

Ensure proper connection of:

* LM35 sensor
* LCD display
* DC motor/fan
* AVR microcontroller pins
